This project is a mini desktop-style Spotify UI clone built as a front-end practice project. The goal is to recreate the look and feel of the Spotify Desktop application using only HTML, CSS, and JavaScript, while improving skills in layout design, component structure, and interactive UI behavior.

Features

- Desktop-style sidebar navigation
- Playlist banner with gradient background
- Track list table with hover interactions
- Music player bar with basic controls
- Clean, modern, and organized layout structure

Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

Project Goals
- Improve front-end styling and layouting skills
- Practice creating complex multi-section UI layouts
- Explore desktop-style UI design patterns for the web

Future Improvements
- Implement an actual audio player using the HTML Audio API
- Add basic state management for selected playlists and tracks
- Improve animations and transitions for a smoother user experience
- Refine responsiveness and overall UX consistency

Status
The project is still under development, and new features or visual improvements may be added over time.
